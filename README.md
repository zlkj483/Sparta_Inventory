# Sparta_Inventory

>Json을 활용하여 인벤토리 및 스테이터스를 구성하고 UI에 연결하여 프로젝트를 진행하였습니다.
>
----

## 🎮 핵심 내용

### 1. 공격력,방어력의 스테이터스를 구현 및 아이템 장탈착에 따라 스테이터스의 변화 및 UI에 적용

### 2. 아이템, 플레이어의 내용을 Json으로 생성 및 struct에 복사하여 매니저 클래스에서 파싱 및 관리

### 3. UIFrameWork를 사용하여 UI전환이 매끄럽게 구성

### 4. 장비 아이템의 경우 장탈착이 가능하고 그 이외에는 불가능하게 코딩

### 5. 인벤토리 슬롯을 프리팹으로 사용하여 아이템 추가에 따라 자동적으로 프리팹클론이 생성되도록 설정

### 6. Resource - Icon 폴더 안의 아이콘이 각 itemName에 따라 자동으로 연결되게 코딩


---

## 🧱 프로젝트 구조 (요약)

> 실제 폴더 구조에 맞춰서 세부 항목은 필요하면 조금씩 수정하세요.

- `Assets/`
  - `Scenes/`  
    - 인벤토리 씬
  - `Scripts/`  
    - `Player/` : 플레이어 데이터, 세이브, 로드, 아이템추가 및 장비 장탈착의 주요 코드
    - `Inventory/` : 아이템 데이터, 인벤토리 및 UI연결
  - `Prefabs/`  
    - 인벤토리 슬롯 프리팹
  - `Json/` : 플레이어, 아이템의 Json
  - `Resources/`
    - `Icons/` : 각 아이템의 아이콘
